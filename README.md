# 🌸 Iris Flower Classification App

This Streamlit web app classifies Iris flowers into Setosa, Versicolor, or Virginica using a trained Random Forest model.

## 🔧 Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy
